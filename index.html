Grand Prix Motorcycle Racing is the premier category of motorcycle road racing. It is made up of three classes.

Class Engine Type
MotoGP 1000cc
Moto2 600cc
Moto3 250cc
  • MotoGP
    • Highest class of GP racing
    • Motorcycles in this class are 1000cc
  • Moto2
    • 2nd level class of GP racing
    • Motorcycles in this class are 600cc
  • Moto3
    • 3rd level class of GP racing
    • Motorcycles in this class are 250cc
